### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

### 二、相关软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户、项目管理员、专家账户、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人中心
5. 公告信息
6. 项目管理



**项目管理员功能：**
1. 登录
2. 修改密码
3. 个人信息
4. 公告信息
5. 项目管理（项目审核）



**专家账户功能：**
1. 登录
2. 修改密码
3. 个人信息
4. 公告信息
5. 项目信息





**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 基础数据管理（公告类型管理、项目类型管理）
4. 公告信息管理
5. 审核日志管理
6. 项目管理
7. 项目管理员管理
8. 用户管理
9. 专家账户管理
